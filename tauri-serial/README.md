# Tauri React Serial Debugger

## Requirements

- Node.js 16+
- Rust 1.70+
- Tauri CLI

## Setup

```bash
# Install dependencies
npm install

# Install Tauri CLI
npm install -g @tauri-apps/cli
```

## Development

```bash
npm run tauri dev
```

## Build

```bash
npm run tauri build
```

## Features

- React frontend
- Rust backend
- Small bundle size
- Native performance
- Secure by default
