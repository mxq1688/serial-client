# C++ Qt6 Serial Debugger

## Build Requirements

- Qt6 (6.2+)
- CMake 3.16+
- C++17 compiler
- Qt Serial Port module

## Build Instructions

```bash
mkdir build
cd build
cmake ..
make -j4
```

## Run

```bash
./LogcatSerial
```

## Features

- Qt6 modern UI
- Serial port communication
- Logcat format support
- Dark theme
- Cross-platform
